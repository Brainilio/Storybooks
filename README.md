# Storybooks

Node.js app using express, mongoDB & google oauth

Entry point:
app.js

To start app (production):
"npm start"

To start app (development):
"npm run dev"

static folder:
./public

views:
./views

routing:
./routes/index.js

config files (mongo uri, port):
./config/..

passport:
./config/passport.js

model for users:
.models/user.js

auth route
./routes/auth.js
